Python 多进程编程

进程和线程：

进程是操作系统分配资源的最小单位，线程是操作系统调度的最小单位。

一个程序，包含一个或多个进程，一个进程包含一个或多个线程。

每个进程在执行过程中，拥有独立的内存单元，而线程再执行过程中，共享内存。

Python中，如果是计算密集型（耗费CPU）建议使用多进程方式，如果是IO密集型（读取操作多的）建议使用多线程。因为Python多线程在CPU上实际上是间断的工作，达不到并发的目的。

Python 多进程

Python在Linux系统下实现多进程，可以调用fork()来实现。fork()函数是一个系统函数。调用一次，有两次返回，因为操作系统再调用该函数时，会自动把当前进程（父进程）复制一份（子进程），然后在父进程和子进程中都返回一次。子进程永远返回0，父进程返回子进程ID。并且父进程要记录下子进程的ID。
